# List Style Position

**Full documentation:** https://warp-ds.github.io/docs/foundations/css-classes/list-style-position

> Typography

Utilities for controlling the position of bullets/numbers in lists.

| Class | Description |
| --- | --- |
| list-inside | list-style-position: inside |
| list-outside | list-style-position: outside |

Control the position of the markers and text indentation in a list using the `list-inside` and `list-outside` utilities.

```html
<ul class="list-inside ...">
  <li>5 cups of coffee</li>
  <!-- ... -->
</ul>
<ul class="list-outside ...">
  <li>5 cups of coffee</li>
  <!-- ... -->
</ul>
```
