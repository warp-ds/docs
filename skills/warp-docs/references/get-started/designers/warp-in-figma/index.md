A quick guide in case you need more libraries.

Go to your desired design file and open libraries.  
  
Hotkeys  
Mac: ⌥option + ⌘command + O  
Win: ctrl + alt + O

![Figma dropdown highlighting the option "libraries"](https://warp-ds.github.io/docs/get-started/figma-dropdown-libraries.jpg)

Search for the desired library and select "Add to file".

![Figma library viewer with “DataViz” as user search input and one result with the Warp - DataViz figma library and the option to add the library to the file."](https://warp-ds.github.io/docs/get-started/figma-libraries-dataviz.svg)

You might want to consider disabling other libraries to not accidentally pick any non-WARP components. All WARP libraries are marked with the "Warp"-prefix.
